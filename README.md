# Local RAG Pipeline for College Essay Feedback

This project implements a complete local RAG (Retrieval-Augmented Generation) pipeline for providing expert feedback on college application essays. It runs fully locally via LM Studio, ensuring privacy and saving costs.

## Requirements

1. Python 3.8+
2. [LM Studio](https://lmstudio.ai/) installed and running a compatible model (e.g., `gemma-4-26B-A4B-it-GGUF/gemma-4-26B-A4B-it-UD-Q4_K_S.gguf`).

## Setup Instructions

### 1. Install Dependencies
Open your terminal in the directory containing this README and run:
```bash
pip install -r requirements.txt
```

### 2. Start LM Studio Server
1. Open LM Studio.
2. Search for and load your model (e.g., `gemma-4-26B` weights).
3. Navigate to the **Local Server** tab (the icon that looks like `<->` or a networked server).
4. Ensure the port is set to `1234`.
5. Click **Start Server**.

### 3. Run the Pipeline

The scripts must be executed in this order:

**Sequence 1: Build the Vector Index**
```bash
python build_index.py
```
This script merges all your local CSV files, filters out invalid/too short essays, generates embeddings using `sentence-transformers` (`all-MiniLM-L6-v2`), and saves them to a local ChromaDB folder named `essay_db`. It prints the progress and an overall summary of the database upon completion.

**Sequence 2 (Optional but Recommended): Grade the Unlabeled Essays**
```bash
python grade_essays.py
```
This script uses your locally running LM Studio model to assign grades and single-sentence feedbacks to any existing essays in the dataset that do not currently have a grade assigned. The newly graded data is saved to `all_essays_graded.csv` piece-by-piece to avoid data loss. **Note: You must have your LM Studio server running for this script to work.**

**Sequence 3: Query for Feedback**
```bash
python query.py
```
This interactive script will prompt you to paste a student essay. Type `END` on a new line and press enter when finished. It will then retrieve the 5 most similar accepted essays from your ChromaDB datastore, inject them into an expert admissions counselor system prompt, and stream you detailed, actionable feedback dynamically generated by LM Studio. **Note: You must have your LM Studio server running.**
